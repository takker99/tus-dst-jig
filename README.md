# tus-dst-jig
卒業研究2025で作った、一面せん断試験機の治具の3Dモデル置き場

OpenSCADで作成している。 rootで`make`を実行すると`renders/`にSTLファイルが生成される。

OpenSCADは安定板ではなく、manifoldを使える最新のnightly buildが必要。
各自でビルドするか、Devcontainerを使うこと。

# Contents

## [羽根付きすき間設定用スペーサー](./dst-spacer.scad)

せん断箱のすき間を設定するためのスペーサー。羽根というかガイドのようなものが上下方向に延びていて、せん断箱を固定する役目も兼ねる。

せん断箱を固定するピンを取り付ける手間を省くために設計した……のだが、湿潤締固め時にせん断箱が浮いてしまうトラブルが発生し、結局ねじ止めが必要になった。本末転倒。
まあ、スペーサー自体は便利なので良しとする。けっこうピッタリはまってかなり固定されるし。

使用する試料の粒径に応じてスペーサーの厚みを変えて印刷する。`spacer_thickness`を変えればよい。

## [締固め用せん断箱カラー](./dst-box-collar.scad)

せん断箱で直接締固め法で供試体を作製するとき、試料があふれ出ないようにガイドするカラー。
本試験機の寸法に合うよう設計している。使うときはfront側とrear側の中央に箇所にクランプで上せん断箱に固定する。

たわまないよう縦方向にリブを入れた。が、本当に必要だったのは横方向のリブだった。水平方向にたわんでしまう。ただこの状態で試しに締め固めてみたが、特に問題なかったのでこのままにした。

## [突固め用平板](./dst-press-guage.scad)

層表面が平坦になるよう突固める平板。せん断箱の内寸に合わせて設計している。

$D_r=75\ \%$程度の乾燥砂であれば、この平板の突起を木槌で叩いて締め固められる。もっと湿潤な試料の場合は締固めエネルギーが不足し、もっと強くたたくと平板が割れるので使用できない。


## [垂直載荷軸を上げたまま固定するストッパー](./dst-rod-stopper.scad)

一面せん断試験機の垂直載荷軸を上げたまま固定するためのストッパー。せん断箱を出し入れするときに使う。


## [垂直載荷軸が上がりすぎないよう制限するストッパー](./dst-box-up-stopper.scad)

操作を誤ってせん断箱を上げきってしまうと、間に入れてある鉛直変位計が強い力で押しつぶされてしまう。
これを防ぐためのストッパー。といってもただの円筒だけど。

天板の高さによって微妙に必要な長さが変わるので、試験機を調整している間はすぐに必要な長さが変わってあまり役に立たない。とはいえ変位計にダメージを与えるよりはマシ。


## [一面せん断試験機の砂を片付けるための漏斗](./dst-sand-collect-funnel.scad)

一面せん断試験機のせん断箱から砂をこぼさずに回収するための漏斗。下せん断箱の台座に取り付けて使う。
試験機の支柱などと干渉しないように形状を工夫している。取り付け部分の長さがやや足りないかも。

試料の種類によってはうまく流れないので、漏斗を叩いて砂を落とす。
漏斗に一気に試料を落としすぎると、台座側から漏れ出る恐れがあるので、少しずつ流す。
